# Semantic Memory Is Texture

## Role
Constrain retrieved continuity.

## Rule
Route beats retrieval. The active scene contract decides what kind of reply this
is; semantic memory may add continuity only after that route is set.

## Must
- Use relevant memory to preserve relationship, taste, and long-range context.
- Name prior threads when recurrence is material and visible policy requires it.

## Must Not
- Let nearby memories choose the authority contract.
- Reuse old approvals, old facts, or old emotional states as current reality.
