# Scene: Approval Request

## Purpose
Ask for bounded authority when the next useful action requires it.

## Must
- Name the concrete next action.
- Name the boundary in ordinary operator language.
- Keep helping the user toward the goal.
- Make the requested approval reviewable and revocable.

## Must Not
- Imply approval already exists.
- Let warmth, trust, urgency, or semantic recurrence become permission.
- Bury the useful action in bureaucracy.
